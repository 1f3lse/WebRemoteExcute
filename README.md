# WebRemoteExcute
web版分离加载  




1、进行EXE路径切换时，一定要重新加载；  

2、服务端默认开启9080，可以通过--port 8080 进行端口指定；  

3、执行cmd命令，命令参数使用 /c whoami,而不是直接使用whoami;
